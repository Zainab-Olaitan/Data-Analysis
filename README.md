
##  Project Overview
This portfolio showcases my restaurant data analytics projects using **Excel**, **Power BI**, and **SQL**. I analyze restaurant sales data, customer behavior, and operational performance to uncover actionable business insights that drive revenue growth and operational efficiency.

##  Objectives
- Analyze sales trends to identify peak hours and high-performing days
- Identify top-selling menu items and revenue contributors
- Understand customer ordering patterns and preferences
- Evaluate staff performance and operational efficiency
- Provide data-driven recommendations for business improvement
- Create interactive dashboards for stakeholders

##  Tools Used
| Tool | Purpose |
|------|---------|
| **SQL** | Data extraction, querying, and database management |
| **Excel** | Data cleaning, transformation, pivot tables, and analysis |
| **Power BI** | Interactive dashboards and data visualization |
